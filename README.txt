Flower Mini App — демо мини-приложение цветочного магазина.

Как открыть:
1. Откройте index.html в браузере.
2. Для проверки телефона: F12 → значок телефона.
3. Для Telegram Mini App загрузите файлы в GitHub Pages и укажите ссылку в BotFather через /setmenubutton.

Что работает:
- главная страница;
- каталог цветов;
- категории;
- поиск;
- карточка товара;
- добавление в корзину;
- изменение количества;
- удаление товаров;
- подсчёт суммы и доставки;
- демо-оформление заказа;
- интеграция с Telegram Web App SDK.

Где менять товары:
Откройте script.js и измените массив products.

Где менять оформление:
Откройте style.css.
